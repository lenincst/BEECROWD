Este repositório reúne soluções para diversos problemas propostos na plataforma Beecrowd.
O objetivo é compartilhar minha evolução em lógica de programação, algoritmos, estrutura de dados e SQL.

As soluções estão distribuídas pelas seguintes categorias da plataforma:

1 - Iniciante – Problemas introdutórios e fundamentos de programação
2 - Ad-Hoc – Simulação, manipulação de datas, lógica geral
3 - Strings – Palíndromos, frequência, LCS, manipulação de textos
4 - Estruturas e Bibliotecas – Filas, pilhas, ordenação, mapas e estruturas clássicas
5 - Matemática – Números, primos, BigInteger, sistemas numéricos
6 - Paradigmas – Programação dinâmica, busca binária, gulosos, backtracking
7 - Grafos – Flood Fill, MST, SSSP, DAG, fluxos, árvores
8 - Geometria Computacional – Pontos, linhas, polígonos e problemas geométricos
9 - SQL – Seleção, inserção, atualização e criação de consultas
